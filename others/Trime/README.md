# Trime（同文输入法）移动端配置

`default.custom.yaml` 是给安卓手机使用的补丁配置，基于雾凇拼音默认配置做最小化覆盖。

## 使用方法

1. 复制 `default.custom.yaml` 到同文用户目录（与 `default.yaml` 同级）。
2. 在同文里执行「重新部署」。
3. 如有个人偏好，再按需修改补丁条目。

## 主要调整

- 候选词数量改为 6，更适合手机屏幕宽度。
- 取消大部分桌面物理键盘翻页绑定，避免与软键盘符号键冲突。
- 中英切换改为 `Shift_L: inline_ascii`，手机上临时输入英文更顺手。
- 方案选单默认不折叠，减少额外点击。
